# PSVT: Pyramid Shifted Window based Vision Transformer for Cardiac Image Segmentation

In recent years, Transformer models have demonstrated significant advancements in natural language processing and computer vision. Various models based on Transformer in cardiac image segmentation tend to integrate with U-Net structures and transform 2D inputs into 3D representations. However, these approaches result in substantial memory consumption and often overlook the essential role of Convolutional Neural Networks (CNNs) in local spatial feature extraction. To tackle these challenges, we introduce Pyramid Shifted-window-based Vision Transformer (PSVT), a novel backbone for cardiac segmentation designed to maintain robust global feature representation while operating on lower-dimensional inputs. By employing a specialized bias generation approach before attention mechanism, we merge the attention module of Swin-Transformer-v2 with CNNs. This fusion results in enhanced computational efficiency and a decrease in memory consumption. Furthermore, we enhance receptive field comprehension by incorporating Depth-Wise convolution (DW-conv) into the Feed-Forward module of attention. Drawing inspiration from numerous U-Net variations, we have restructured the Patch Merging and Patch Expanding modules, incorporating transposed convolution techniques to seamlessly fuse multi-scale features into the classification head. Our experimental results clearly demonstrate the superiority of our model. It outperforms state-of-the-art cardiac segmentation models across three small-sample cardiac datasets, including ACDC, MMWHS-CT, and LASEG-2013.
